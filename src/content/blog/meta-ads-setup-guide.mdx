## 01  먼저 이해할 것 — 메타 광고의 3단계 구조

본격적인 메타 세팅에 앞서, 메타 광고가 어떤 위계 구조로 작동하는지 반드시 이해해야 합니다.

| 구조 단계 | 영문 명칭 | 핵심 설정 항목 | 한 줄 요약 |
| --- | --- | --- | --- |
| 1단계 | 캠페인 (Campaign) | 광고 목표, 캠페인 예산(CBO) | "무엇을 달성할 것인가" |
| 2단계 | 광고 세트 (Ad Set) | 타겟, 예산, 위치, 일정, 노출 위치 | "누구에게 보여줄 것인가" |
| 3단계 | 광고 (Ad) | 소재(이미지/영상), 카피, 링크, CTA | "무엇을 보여줄 것인가" |

이 3단계 구조는 모든 메타 광고의 뼈대입니다. 캠페인 1개 아래에 여러 광고 세트, 광고 세트 1개 아래에 여러 광고를 배치하는 'A/B 테스트' 구조로 운영하는 것이 정석입니다.

## 02  비즈니스 자산부터 분리한다

메타 세팅의 첫 단추는 개인 계정이 아닌 비즈니스 자산을 분리하는 것입니다. 개인 프로필로 광고를 돌리면 계정 정지 시 모든 자산을 한 번에 잃습니다.

| 자산 | 역할 | 필수 여부 |
| --- | --- | --- |
| 비즈니스 관리자(Business Manager) | 모든 자산의 컨트롤 타워 | 필수 |
| 광고 계정(Ad Account) | 결제 및 캠페인 운영 단위 | 필수 |
| 페이스북 페이지 | 광고 게재 주체 | 필수 |
| 인스타그램 계정 | 인스타 노출용 (페이지와 연동) | 권장 |
| 메타 픽셀(Pixel) | 웹사이트 전환 추적 코드 | 강력 권장 |
| 결제 수단 | 신용카드 또는 계좌 | 필수 |

## 03  메타 픽셀 — 전환 추적의 핵심

픽셀 없이 메타 세팅을 마쳤다면 절반만 완성한 것입니다. 픽셀은 웹사이트 방문자의 행동(구매, 장바구니, 결제 등)을 추적해 알고리즘에게 '학습 데이터'를 제공합니다.

| 이벤트명 | 추적 행동 | 활용 캠페인 |
| --- | --- | --- |
| PageView | 페이지 조회 | 트래픽 |
| ViewContent | 상품 상세 조회 | 리타겟팅 |
| AddToCart | 장바구니 담기 | 전환 |
| InitiateCheckout | 결제 시작 | 전환 |
| Purchase | 구매 완료 | 전환(핵심 KPI) |
| Lead | 문의/DB 제출 | 잠재고객 확보 |

설치 방법은 ①수동 코드 삽입, ②구글 태그 매니저(GTM), ③파트너 연동(카페24·아임웹 등) 세 가지가 있으며, 코드를 잘 모른다면 GTM이나 파트너 연동을 추천합니다.

![메타 픽셀 표준 이벤트 - PageView·Purchase·Lead](/images/blog/meta-ads-setup-guide/4.png)

## 04  캠페인 목표 — 알고리즘에게 방향을 알려주는 단계

메타 세팅에서 가장 흔한 실수가 목표를 잘못 선택하는 것입니다. 목표는 알고리즘이 누구를 찾아갈지 결정합니다.

| 캠페인 목표 | 추천 상황 | 최적화 방향 |
| --- | --- | --- |
| 인지도(Awareness) | 브랜드 노출 극대화 | 도달·노출 |
| 트래픽(Traffic) | 웹사이트 방문 유도 | 클릭(CPC) |
| 참여(Engagement) | 게시물 반응·메시지 | 좋아요·댓글·DM |
| 잠재고객(Leads) | DB·문의 수집 | 양식 제출 |
| 앱 홍보(App) | 앱 설치·실행 | 앱 이벤트 |
| 판매(Sales) | 구매 전환 극대화 | Purchase 전환 |

매출이 목적이라면 반드시 '판매(Sales)' 목표를 선택해야 알고리즘이 구매 가능성이 높은 사람을 찾아갑니다.

![메타 광고 캠페인 목표 - 인지도·트래픽·판매](/images/blog/meta-ads-setup-guide/5.png)

## 05  타겟 설정 — 메타 세팅의 정밀도를 결정한다

광고 세트 단계에서 진행하는 타겟 설정은 메타 세팅의 정밀도를 결정합니다.

[ 타겟 유형 비교 ]

| 타겟 종류 | 설명 | 주요 활용 |
| --- | --- | --- |
| 핵심 타겟 | 관심사·인구통계·행동 기반 | 신규 고객 발굴 |
| 맞춤 타겟 | 픽셀·고객리스트·영상 시청자 | 리타겟팅 |
| 유사 타겟 | 기존 고객과 닮은 신규 모수 | 확장(Scale-up) |

[ 추천 타겟 운영 전략 ]

| 단계 | 타겟 구성 | 목적 |
| --- | --- | --- |
| 초기 | 관심사 + 광범위(Broad) | 데이터 수집 |
| 성장기 | 유사 타겟 1~3% | 효율적 확장 |
| 리타겟 | 사이트 방문자·장바구니 이탈자 | 전환 마무리 |

[ 예산·입찰 설정 ]

| 항목 | 옵션 | 추천 |
| --- | --- | --- |
| 예산 방식 | CBO(캠페인 예산) / ABO(광고세트 예산) | 초기엔 ABO, 안정화 후 CBO |
| 예산 유형 | 일일 예산 / 총 예산 | 일일 예산 |
| 입찰 전략 | 최저 비용 / 비용 한도 / 입찰가 한도 | 초기엔 최저 비용 |
| 노출 위치 | 자동 노출 위치 / 수동 선택 | 자동(Advantage+) |

요즘 메타 알고리즘은 정교한 세분화보다 광범위 타겟(Advantage+ 타겟)에 데이터를 몰아주는 방향이 효율적인 경우가 많으니, 초기 메타 세팅 시 너무 좁게 잡지 않는 것이 좋습니다.

## 06  광고 소재와 성과 점검

마지막 단계는 실제 사용자가 보는 광고입니다. 카피는 ①후킹(첫 문장) → ②문제 제기 → ③솔루션 → ④CTA 구조로 작성하고, 광고당 소재 2~4개를 넣어 메타가 자동으로 우수 소재를 찾도록 하는 것이 좋습니다.

[ 소재 규격 가이드 ]

| 소재 규격 | 권장 사이즈 | 비고 |
| --- | --- | --- |
| 피드 정사각형 | 1080 × 1080 | 범용성 최고 |
| 스토리·릴스 | 1080 × 1920 | 세로 풀스크린 |
| 영상 길이 | 15초 이내 | 초반 3초가 승부 |
| 텍스트 비율 | 이미지 내 20% 이하 권장 | 가독성 확보 |

[ 핵심 성과 지표 ]

| 지표 | 의미 | 점검 포인트 |
| --- | --- | --- |
| CTR(클릭률) | 노출 대비 클릭 비율 | 1% 미만이면 소재 점검 |
| CPC(클릭당 비용) | 클릭 1회당 광고비 | 산업 평균과 비교 |
| CPA(전환당 비용) | 전환 1건당 비용 | 최종 수익성 판단 |
| ROAS(광고수익률) | 광고비 대비 매출 | 3 이상이면 양호 |
| 빈도(Frequency) | 1인당 노출 횟수 | 3 초과 시 피로도 ↑ |

[ 자주 하는 실수와 해결책 ]

| 실수 | 결과 | 해결책 |
| --- | --- | --- |
| 픽셀 미설치 | 전환 추적 불가 | 발행 전 픽셀 검증 |
| 트래픽 목표로 매출 기대 | 클릭만 많고 구매 없음 | 판매 목표로 변경 |
| 학습 중 잦은 수정 | 학습 초기화 반복 | 3~4일 유지 후 판단 |
| 타겟 과도한 세분화 | 모수 부족·단가 상승 | 광범위 타겟 활용 |
| 소재 1개만 운영 | 테스트 불가 | 소재 2~4개 배치 |

## 마치며 — 탄탄한 세팅이 광고 성과의 기반이 됩니다

메타 광고는 '돈을 넣고 켜는' 시스템이 아니라, 정교한 메타 세팅 위에서 학습하고 최적화되는 머신러닝 엔진입니다.

처음에는 복잡해 보여도, 한 번 구조를 제대로 잡아두면 이후 캠페인 확장은 훨씬 수월해집니다. 탄탄한 메타 세팅은 결국 광고 성과의 가장 든든한 기반이 됩니다.

온케팅은 메타 비즈니스 계정 세팅부터 픽셀 설치, 캠페인 운영과 성과 점검까지 함께 합니다. 메타 광고를 처음 시작하거나 성과가 정체됐다면 언제든 문의 주세요.
