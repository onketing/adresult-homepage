### 1-1. 무엇을 하는 회사인가
**병원·의원 전용 마케팅 대행사.** 상품에 병원을 맞추지 않고, **진료과별 1:1 맞춤 전략**으로 신규 환자(신환)를 늘립니다. 누적 1,272곳, 전국 200여 곳과 함께합니다.

핵심 서비스:
- **AIO마케팅 (AI 검색 노출·GEO)** — ChatGPT·Gemini·Perplexity·구글 AI 검색에서 병원이 추천되게. (`/services/aio`)
- **끝장숏폼 (숏폼·릴스·쇼츠 올인원)** — 조회수가 아닌 예약을 만드는 영상, 현직 인플루언서가 기획·촬영·편집·관리까지. (`/services/shortform`)
- **바이럴·유튜브·블로그** — 카페 바이럴, 유튜브 콘텐츠, 검색 유입 콘텐츠.

### 1-2. 타겟 (진료과)
피부과 · 성형외과 · 정형외과 · 통증의학과 · 치과 · 한의원 (진료과 랜딩 6종, `/specialty/[slug]`). 1차 의사결정자는 각 병원의 **원장·대표원장**.

### 1-3. 차별점 (사이트 카피 기준)
1. **결과로 말합니다** — 노출·조회수가 아닌 신환·예약·매출로 평가. "결과에 집착합니다."
2. **1:1 병원 맞춤 전략** — 상품에 병원을 맞추지 않고 병원마다 전략을 설계 (Win-Win).
3. **업력·데이터** — 11년, 누적 1,272곳, 최장수 고객 132개월, 사내 지식뱅크 3,800여 개, 애드리절트TV 500여 개 인사이트 영상.
4. **의료광고법 준수** — 의료법 §56을 수시 학습하며 규정을 지키면서 성과를 만듦.

## 3. 톤·매너 가이드

### 3-1. 원칙
- **결과 지향** — 노출·조회수보다 신환·예약·매출. 검증 가능한 수치(11년·1,272곳·132개월·3,800개·40~333% 증가·매출 7~8배)로 자신감을 표현.
- **단정형 마침** — "~합니다 / ~됩니다".
- **한 문장 한 의미** — 짧게 끊습니다.
- **의료광고법 안전** — 절대적·단정적 표현("최고", "100% 보장", "완치", "부작용 없는") 금지. 부작용·주의사항은 상담 안내로. (규정 상세: `/regulation`, `docs/04-pages.md`)

### 3-2. 자신감의 출처 (사실 진술)
| 출처 | 표현 |
|---|---|
| 업력 | "업력 11년" |
| 누적/현재 고객 | "누적 1,272곳 / 전국 200여 곳" |
| 최장수 고객 | "132개월째 함께" |
| 지식 자산 | "사내 지식뱅크 3,800여 개 / 애드리절트TV 500여 개 영상" |
| 파트너십 | "T-Map 공식 파트너사" |
| 실제 성과 | "신환 40%~333% 증가, 매출 7~8배" (익명 고객사) |

### 3-3. 의료광고법 유의 (필수)
콘텐츠·카피 작성 시 의료법 §56 사전심의 기준을 지킵니다. `/regulation` 페이지의 진료과별 허용/금지 표현을 참고하고, 위반 소지 카피는 발행 전 점검합니다. **법령 인용·심의 관련 최종 판단은 규정 담당자 검수를 거칩니다.**

## 4. 디자인 톤

### 4-1. 컬러 (테마 = 빨강)
| 용도 | 값 |
|---|---|
| 강조 1 (밝은 레드) | `#ef3c39` — eyebrow, 강조 span, ring, badge |
| 강조 2 (진한 레드) | `#e11d29` — CTA·버튼·탭 active, 레드 풀폭 밴드 |
| 딥 레드/다크 | `#3a0509` · `#0d0202` · `#140505` · `#0b1220` — 히어로/다크 CTA/규정 섹션 |
| 노란 CTA 버튼 | `#ffe14d` / `#FAE100` (전화·카카오 강조) |
| 본문/제목 | `#0a0a0a` (제목) · `text-slate-600`(본문) |
| 배경 분리 | `bg-white` ↔ `bg-slate-50` 교차 |

### 4-2. 시각 리듬
밝음(white/slate-50) 연속 사이에 **레드 풀폭 밴드(`#e11d29`)** 를 앵커로 반복. 서비스·진료과 페이지는 "white → 레드 밴드 → white → 레드 밴드" 패턴으로 리듬을 만든다. 히어로/최종 CTA는 딥 다크 레드.

### 4-3. 타이포그래피
- **Eyebrow**: `font-bold text-[#ef3c39] text-sm uppercase tracking-[0.25em]` (영문 라벨: Excellent Company, ADRESULT YOUTUBE, How we do 등)
- **H1·H2**: `font-extrabold tracking-tight` 굵고 단정
- **강조 span**: `text-[#ef3c39]` 또는 `text-[#e11d29]`
- **손글씨 강조**: `--font-hand`(Nanum Pen Script) — CustomStrategy 등 특정 강조
- **본문**: `leading-relaxed break-keep`

### 4-4. 컴포넌트 패턴
- 회사소개 하위 페이지: `CompanyHero`(영상) + `CompanyTabs`(4탭) 조합.
- 목록 페이지(성공사례·후기): `PaginatedCards`.
- 섹션 진입 모션: `Reveal`. 마퀴: `react-fast-marquee`.
- 섹션 제목은 공용 컴포넌트 없이 eyebrow `<p>` + `<h2>` 마크업을 직접 작성 (docs/03-components.md "섹션 공통 패턴").

## 5. 사이트 구조

### 5-1. 라우트 트리
```
/                       메인
/about /history /ceo /people   회사소개
/services/aio           병원 AIO·AI검색(GEO)
/services/shortform     끝장숏폼
/specialty/[slug]       진료과 6종 (피부과·성형외과·정형외과·통증의학과·치과·한의원)
/cases  /cases/[slug]   성공사례
/cases/create           관리자 케이스 에디터
/reviews                고객후기
/regulation             의료광고 규정 가이드
/faq                    자주 묻는 질문
(/contact → Tally 폼 리다이렉트)
```

### 5-2. 글로벌 네비게이션
```
회사소개 ▾ / 마케팅 ▾ / 고객사례 ▾ / FAQ      [회사소개서] [문의하기]
```
- 회사소개 ▾: ABOUT / HISTORY / CEO / PEOPLE
- 마케팅 ▾: AIO마케팅 / 끝장숏폼
- 고객사례 ▾: 성공사례 / 고객후기

### 5-3. 동선 의도
```
메인(공감·수치·후기·사례) → 회사소개(About/History/CEO/People)
     → 서비스(AIO / 끝장숏폼) → 진료과 랜딩(specialty)
     → 성공사례·후기(증거) → 문의(Tally/전화/카카오)
```
진료과 랜딩은 진료과 키워드로 유입된 원장을 서비스(AIO·숏폼)와 문의로 연결하는 SEO/AEO 진입점.
